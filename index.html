LocalFreeWeb

Add location

LocalFreeWeb is a Code for San Francisco initiative that promotes internet enabled computers available to EVERYONE.

How it works

Get access to a free internet enabled computer in 3 easy steps.
1

Text any MUNI bus Stop ID to
415-528-3226.

2

Receive text message with information on computer
locations near you.

3

Get FREE access to internet enabled computers.

Add a new location

Enter information for another location that provides free internet

About

LocalFreeWeb is brought to you by Code for San Francisco, a Code for America brigade.

Get involved

Want to help spread the word about LocalFreeWeb? Post a flier near a San Francisco Muni Bus Stop!
Interested in setting up LocalFreeWeb for your city? Fork the repo: github.com/sfbrigade/localfreeweb-sms-api
